# PWA Icons

This directory should contain PWA icons in the following sizes:
- icon-72x72.png
- icon-96x96.png
- icon-128x128.png
- icon-144x144.png
- icon-152x152.png
- icon-192x192.png
- icon-384x384.png
- icon-512x512.png

## Generating Icons

To generate icons from a base image (e.g., logo.png), run:

```bash
python generate_pwa_icons.py logo.png
```

The input image should be at least 512x512 pixels and preferably square.

## Temporary Placeholder

If you don't have icons yet, you can create simple placeholder icons using any image editor or online tool. The icons should:
- Be square (same width and height)
- Have a transparent or solid background
- Represent your application/brand
- Be optimized for web (PNG format)
